# `marv setup`

Initialize `~/.marv/marv.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
marv setup
marv setup --workspace ~/.marv/workspace
```

To run the wizard via setup:

```bash
marv setup --wizard
```
