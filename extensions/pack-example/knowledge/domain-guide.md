## Knowledge File Format

- Use Markdown (.md) or plain text (.txt) files
- Place files in the `knowledge/` directory of your pack
- Files are automatically chunked and indexed for semantic search
- Keep each file focused on a specific topic for better retrieval
- Use clear headings to improve chunk quality
